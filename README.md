# SSM-curd
第一个SSM整合（CRUD）项目

- 主要功能
    - 分页
    - 数据校验
        - jQuery前端校验 + JSR303后端校验
    - ajax
    - Rest风格的URI
        - 使用HTTP协议请求方式的动词，来表示对资源的操作（GET（查询），POT（新增），PUT（修改），DELETE（删除））
        
- 主要技术
    - 基础框架-ssm
    - 数据库-MySql
    - 前端框架-bootstrap
    - 项目的依赖管理-Maven
    - 分页-PageHelper
    - 逆向工程-MyBatis Generator
